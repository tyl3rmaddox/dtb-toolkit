Introduction

The purpose of this guide is to cover the basics of building, exporting, and validating a Digital Talking Book (DTB) using Hindenburg Audio Book Creator (Hindenburg ABC) based on National Libary Service (NLS) Specifications. This guide assumes the user has access to wav file(s) containing a narrated text, an Open Package Format (opf) file with appropriate metadata, and tools for encrypting and validating an unencryped DTB in compliance with NLS specifications.

1. Create a New Folder
  • Create an empty folder in Production which will be used to store your build session and exported DTB files

  • The folder must be named with only the numerals of the DB number for the sake of validation. i.e. 88111 not DB88111 or 88111_dont_do_this
2. Save Session
  1. Open Hindenburg NLS ABC and select File > Save As..

  2. Navigate to and select the newly created DB# folder

  3. Save as "session" and remember to frequently save (CTRL+S) during the build process
3. Enter Metadata
  1. File > Properties > Info (default tab)

  2. Scroll to the bottom, select import, select corresponding .opf file using DB#. Ensure you don't change Language and Multimedia Type drop boxes as you scroll. (default to Language = English-United States and Multimedia Type = Audio, NCX)

  3. Fill out..
    1. Narrator
    2. Produced date (must always current date of build/validation, change for any reruns)
    3. (For revisions) Revision date, revision number, and revision description
    4. Copy DTB Producer to Recording Agency line
4. Set Default Styles
  1. While in Properties, select the Styles tab

  2. Under the Class column double click the desired field to set default attributes corresponding to most frequently used attribute for each level (H1 up to H6)

    • As you type in class attribute a drop down menu should appear with auto filled attribute options to select ensuring correct spelling.
    • H1 meaning level 1 as indicated with the marks on the wav file i.e. H1 set to “chapters” for a 1 level book or set to “parts” for a 2 level book
    • You will most likely never need to set these to singularly used attributes i.e. cast, acknowledgement/c, concluitem
  3. For books with simple headings that are numerals (1,2,3,4 etc) you can use the text column to set them to auto number by selecting the header field and inputing #.
5. Import WAV Files

Ensure your playhead is at 00:00:00 and select Import from the Tool Bar. Highlight all .wav files to be used in the build from the NLS books folder and wait a few moments for the workspace to process the incoming audio.

6. Set DTB navigation

While the marks we use in studio recorder hold no bearing on a Hindenburg build they are still visible and navigable in the workspace. This will be helpful when setting headings, notes, and side breaks. To navigate between these visual “cues” you can use Tab or Page Up/Down. This jumps you to all cues in the file including level marks, generic marks, and side breaks. You will recognize which level a heading should be marked based on the number of the cue. (a 1 mark will correspond to level 1 navigation)

  1. Headings:
    • Click and drag above or at the bottom of the WAV bar for the duration of the heading you wish to set. (in BWP, this would be the space from a navigation mark to its closing generic mark) A green arrow will indicate the beginning of heading audio and a red arrow will indicate the end. You can adjust these freely before giving the mark a value. To preview your work, press Shift + Space concurrently to isolate the selection.
    • To clear a selection, press escape or double click in the workspace.
    • Do not overlap headings. For example, book title and author should have some separation even though they are alway sequential.
    • Once you are pleased with the heading, choose the appropriate attribute from the Book Bar (Title, Author, Annotation, Close, H1-H6, etc.) or use keyboard shortcuts as seen in the Keyboard Shortcuts section.
    • Once set, you are free to assign your headings values at any point during the build based on preference in Manuscript View. You can do this as you go or set them collectively at the end.
    • If you would like to delete a heading that you have already set, you can do so by pressing Delete while the heading is selected in Manuscript View. Note: Ensure that the heading is in Select Mode (yellow) and not in Edit Mode (blue) when deleting.
  2. Specials without Navigation (Readers Notes/Sidebars/Boxes):
    • These marked but reference free specials will be identified by the same numbers as seen in Studio Recorder (8 for beginning and 7 for closing).
    • Set in the same manner as headings -- select the beginning of a note and drag to the end (from the 8 mark to the 7)
    • Choose the corresponding attribute from the Book Bar
  3. Specials with Navigation (Foot notes/Explanatory notes):
    • Set the playhead at the beginning of the Note Reference and select Note-Ref from the Book Bar. (the first 8 of the pair)
    • Highlight the entirety of the Note Content (from the second 8 to the 7) and select Note-Content from the Book Bar
    • If your Note References require numbering, you can do so in Manuscript View in the same way you would a heading. If there is no number associated with the note, it will be given the generic notation by default.
  4. Segmented Lists:
    • Navigate to each 7 imported from Studio Recorder and drop a Segment mark on each. The quickest way to do this is to tab and click on sement until you find the next mark that is not a 7.
    • Be sure to adjust your 7 marks before importing to make this process quick and easy
  5. Side Breaks
    • Side Break Markers are automatically generated at 90 minute intervals when WAV files are importing into the software.
    • As you do markup, you will Tab to a split in WAV files corresponding to the end of a side.
    • From here, find the nearest Side Break Mark (should be located around 1-2 ahead of the side break) and drag it to the silence between sides. Note: Because the audio no longer skips to the generic marker located at the beginning of a WAV file, ensure there isn't an unnatural amount of silence between sides.
7. Fill Out Text Values

If you choose to fill out your navigable headings after markup, be sure to use Escape and F2 to toggle between Edit Mode (blue) and Select Mode (yellow). Do not press Enter to finalize an entry as it will create a blank navigable heading that is easy to miss. In edit mode, you can edit the text label to match the text of the book, then press Escape to switch to Select Mode in order to navigate to the next heading. Press F2 to reenter Edit Mode and highlight the entire heading for entry.

8. Change Class Attributes

Most class attribute will be filled in according to the defaults that you set in the Styles Tab (File > Properties > Styles), but there will be certain class attribute that will need to be changed during or after the build to reflect DTB navigation. (such as when Acknowledgments falls on the same level as chapters)

  • Make sure your non-default class attributes are set to the correct level (H1, H2) even if they appear in manuscript mode as the default attribute. (Part, Chapter) If they are incorrect just right click the heading (or press the Function key) and hover over the second to last line and select the correct level from the menu of selections.
  • You can edit class attributes at any time and while in Select or Edit Modes
9. Export Completed DTB

When you have finished editing your book and you are ready to run the error/spell check and validate, click on the white arrow to the right of Export in the menu bar up top and select Export NLS DTB from the drop down menu. Select the correct DB folder in Production and begin export.
